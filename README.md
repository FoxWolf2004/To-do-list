# 📝 ToDo List — управляй задачами с комфортом

Простое, но функциональное приложение для ведения списка дел с тремя статусами, сохранением в `localStorage` и адаптивным интерфейсом.

## 🚀 Демо

👉  [Смотреть демо](https://foxwolf2004.github.io/To-do-list/)

## 📌 Возможности

- ✅ Добавление задач с текстом (textarea)
- 📋 Три статуса: **Не завершено**, **В процессе**, **Завершено**
- 🔄 Изменение статуса через выпадающий список
- 🗑️ Удаление ненужных задач
- 💾 Автоматическое сохранение в `localStorage` — данные не пропадают после перезагрузки
- 📊 Счётчик задач по каждому статусу
- 📱 Адаптивный дизайн для мобильных устройств и ПК

## 🛠️ Технологии

- **HTML5** — семантическая разметка
- **CSS3** — флексы, гриды, адаптив, кастомные цвета под статусы
- **JavaScript (ES6+)** — работа с DOM, массивами, событиями, `localStorage`

## 🎨 Особенности дизайна
Задачи в разных статусах имеют цветовую индикацию:

🔴 Не завершено — красноватый фон

🔵 В процессе — голубоватый фон

🟢 Завершено — зеленоватый фон

Карточки задач  скруглённые углы и плавные переходы

Кнопка удаления с иконкой (кастомная)
